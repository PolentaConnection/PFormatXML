# Changelog

All notable changes to PFormatXML are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0] - 2025-01-15

### Added
- Initial public release
- Format and pretty-print XML files from the command line
- Configurable indentation (spaces or tabs)
- Optional backup before overwrite
- Recursive processing for directories and wildcard patterns
